<b>

Bold text is text that is darkened to emphasize the content.

Usage information

This element is part of the highlighting domain. Use this element only when a more semantically appropriate element is not available. For example, for specific items such as GUI controls, use the <uicontrol> element.

Formatting expectations

Although rendering is left up to implementations, processors typically apply bold highlighting to the contents of the <b> element.

Specialization hierarchy

The <b> element is specialized from <ph>.

Attributes

The following attributes are available on this element: Universal attribute group and outputclass.

Example

<p><b>STOP!</b> This is <b>very</b> important!</p>
